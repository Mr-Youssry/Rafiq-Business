# Turkey - Ecosystem (Network & Support Infrastructure)

## Section Score: 4/5

Turkey has a surprisingly robust startup ecosystem with strong government support, active angel networks, and multiple accelerator programs. The ecosystem is Istanbul-centric with emerging activity in Ankara. While EdTech is not the dominant investment sector, there are clear pathways for building network connections.

### Government-Backed Programs

**TÜBİTAK BİGG (Primary Government Program)**

| Element | Details |
|---------|---------|
| Grant Amount | 200,000 TL (~$6,000) |
| Equity | None (basic) or 3% (investment) |
| Implementing Orgs | 149 across Turkey |
| Application | Rolling, through implementing organizations |
| Eligibility | Tech startups, open to foreigners via local entity |

**Additional Programs:**

- **KOSGEB**: SME support programs
- **Technopark Incentives**: Tax exemptions, subsidized space
- **TÜBİTAK 1512**: Entrepreneurship support program

### Application Strategy

**Recommended Path:**
1. Apply to **Startupbootcamp Istanbul** or **KWORKS**
2. Simultaneously pursue **TÜBİTAK BİGG** through implementing org
3. Use accelerator network for VC introductions
4. Consider technopark registration for ongoing benefits
